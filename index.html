    Welcome to LegacyXperia Project homepage.

    LegacyXperia Project is an unofficial build of CyanogenMod 10.1 for legacy Sony Ericsson msm7x30 devices. It is an effort to join all developer forces to bring a stable cm-10.1 rom on these devices which are dropped from official CM.

    Supported devices:

    Currently only mogami devices are supported. zeus devices don't have a 3.4 kernel port yet so it's not possible to support them, no active developer has this device.

    List of supported devices & links to xda support & development threads:

    • Xperia Arc/Arc S (anzu)
    • Live with Walkman (coconut)
    • Xperia Neo V (haida)
    • Xperia Neo (hallon)
    • Xperia Pro (iyokan)
    • Xperia Mini Pro (mango)
    • Xperia Active (satsuma)
    • Xperia Mini (smultron)
    • Xperia Ray (urushi)
